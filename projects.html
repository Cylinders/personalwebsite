Voice Pilot: April 2024 - Present



Link to Github Repository
Go Back to All Projects

Voice Pilot is an innovative project that harnesses the capabilities of contemporary large language models (LLMs) to create an intuitive voice-controlled interface for computers. Drawing inspiration from the fictional AI assistant Jarvis, Voice Pilot enables users to interact with their computers through natural speech, seamlessly translating verbal commands into precise computer movements. This system simulates the functionality of a mouse and keyboard, allowing for hands-free control and enhancing accessibility. Leveraging the power of PyAutoGUI for automation and the GPT API for advanced language processing, Voice Pilot transforms the way users engage with their digital environments, offering a fluid and efficient alternative to traditional input devices.

Technologies used and Awards Won:

Gemini API

SpeechToText

PyAutoGui

TKinter

NJTSA Third Place Software Development.

HumanLikeTyper: February 2024



Link to Github Repository
Go Back to All Projects

Vihas Veggalam and I developed a sophisticated application that simulates human typing with exceptional accuracy and speed. Initially crafted as a Command-Line Interface (CLI) application in Python, we transformed it into a versatile Graphical User Interface (GUI) application using Java, leveraging the Java Robot class to achieve typing speeds of 350-400 words per minute while mimicking common typing mistakes to evade detection by CAPTCHAs and biometric algorithms. This innovative project highlights our expertise in creating realistic software solutions and our ability to seamlessly transition between different programming languages and platforms.

Technologies used:

Java Robot

Pyautogui
